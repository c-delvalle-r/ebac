Definición de un modelo determinísitico de un cálculop de costos¶

La empresa est considerando tomar un proyecto. Tienen un presupuesto para completar el proyecto de $10,000. Corrern simulaciones de los diferentes costos que tienen previstos con funciones de probabilidad que validaron expertos en el tema. Si la probabilidad de que el proyecto entre dentro de su presupuesto es de .60 o ms, tomarn el proyecto, de lo contrario no lo realizarn. Los costos que tienen en cuenta son los siguentes:

  • Costo de construccin
  • Costo de materias primas
  • Costo de planeacin
  • Costo de transporte
  • Costo de mano de obra

El costo total del proyecto es la suma de los 5 costos mencionados anteriormente

def roll_dice():
    costo_construccion = np.random.triangular(1500,2000,3000)
    costo_materias = np.random.uniform(1500,2250)
    costo_planeacion = np.random.normal(loc=1900, scale=500)
    costo_transporte = np.random.triangular(1500,2000,2700)
    consto_labor = np.random.uniform(1800,2200)

    return costo_construccion, costo_materias, costo_planeacion, costo_transporte, consto_labor

Insights 2:¶

  • Como se puede ver en el anlisis de sensibilidad, el cambio que result en el mayor efecto hacia la probabilidad de que el proyecto entre dentro del presupuesto asignado, fue el cambio en el promedio del costo de planeacin el cual presenta una distribucin normal

Conclusiones finales:¶

  • La simulacin es sumamente efectiva para crear una cantidad alta de escenarios dadas ciertas variables siempre y cuanto se conozca la distribucin que siguen esas variables
  • Conocer la distribucin de las variabes es un tema completamente a parte puesto que se requiere de obtener suficientes muestras de las mismas y correr otro tipo de tests estadsticos para determinar cul distribucin es a la que ms se asemejan los datos
  • Todo el esfuerzo que requiere la obtencin de datos y la realizacin de pruebas para la determinacin de la distribucin probabilstica que siguen las variables puede o no valer la pena, dependiendo especficamente de cada caso, la disponibilidad de datos histricos, la dificultad de observacin del evento que genera la variable que se est tratando de simular, entre otras cosas
